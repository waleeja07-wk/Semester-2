# SEMESTER-2

## 📖 About This Repository

This repository is a complete archive of my second semester coursework. The goal is to keep everything from the semester backed up and organized in one place — so that all my work is safe, and easy to look back on whenever needed.

## 🎯 Purpose

- To maintain a **single source** for all semester-related academic work.
- To keep a **history of progress** for every subject using Git, including diffs that show how each assignment/task evolved over time.
- To make **revision and reference easier** before exams or future projects by having everything in one organized place.
- To ensure **no work is lost** — all files, drafts, and final submissions are backed up here.

## 📦 What's Included

This repo contains **all the content from my semester**, including:

- ✅ Assignments for every subject
- ✅ Projects
- ✅ Lab tasks and their pdfs
- ✅ Diff files tracking progress for each subject's work
- ✅ Any additional notes or resources used throughout the semester

Nothing has been left out — every subject taught this semester has its corresponding folder with the complete set of academic work pushed to this repo.

## 📁 Structure

Each subject has its own folder containing the assignments, projetcs, and lab tasks completed for that subject, along with diff history showing how the work progressed.

## 🚀 Usage

This repository is primarily for personal reference and backup. Feel free to browse through it, but note that the content reflects my personal coursework and problem-solving approach.

```bash
git clone https://github.com/waleeja07-wk/Semester-2.git
```
